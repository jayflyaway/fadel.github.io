Other projects

  • mp (R package)

    I have developed, jointly with Francisco Fatore, the mp package for R. It contains implementations of some multidimensional projection techniques, useful for visualizing high dimensional data. Hoping to achieve very good performance, many techniques are written in C++ using armadillo (with Rcpp and RcppArmadillo).
